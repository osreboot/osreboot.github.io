"Osmium" Game Prototype

For-Fun | Solo Project

Java

OpenGL

GLFW

GLSL

LWJGL3

A realtime multiplayer 2D space combat simulator based on Newtonian physics. Featuring ship trajectory projection and maneuver planning, real-life missile tracking algorithms (proportional navigation), and automated turrets capable of leading target velocity and acceleration!

A modified ECS architecture allows for a clean implementation of rollback netcode (server-client synchronization with client prediction and correction). The project is built directly with pure Java sockets (TCP) and does not rely on helper libraries for networking. Deterministic game physics enables input-only communication and allows for hundreds of unique bullet and missile entities at no additional networking expense.

Custom Level for Half-Life: Alyx link

 For-Fun | Solo Project

Source 2

Lua

Substance Designer

Blender

Featured by Valve's guest writer Craig Pearson in the one-year anniversary mod showcase!

Emergent is a complete gamemode built into an original map layout. The engine's scripting system has been utilized to drastically alter the standard gameplay rules of Half-Life: Alyx. The map has over 9,000 current subscribers and over 200 ratings, with a 95% positive ratio.

Though it is generally inspired by CoD: Zombies, Emergent is built using fully original assets and mechanics. These components are designed to compliment the specific systems of Half-Life: Alyx, and are not built simply to copy the map's inspiration material.

The map features a complex point economy system, powerups that drop from killing enemies, new physics interactions between props and hostile entities, machines that vend items, puzzles to unlock said machines, and a wave-based difficulty scaling system.

"Nucleocide" Game Prototype link

 For-Fun | 5-Person Team Project

Java

OpenGL

GLSL

LWJGL2

Nucleocide is a top-down multiplayer shooter concept that explores light-based combat.

My primary role was to construct the game's graphical rendering system. Built using a Java-based OpenGL binding (LWJGL), the system features dynamic pixel-perfect shadows and intricate world material lighting.

A deferred rendering architecture simplifies the code footprint and allows both entities and tiles to exhibit a range of lighting effects. World tile visual data is supplied in the form of color, normal, metalness, and occlusion textures. Dynamic entities can supply additional emission and displacement textures. A custom GLSL fragment shader ingests this data over several rendering passes and combines it to produce the final display image.
